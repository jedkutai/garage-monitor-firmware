# BME280 Wiring

## Wiring

```text
BME280 VIN → STM32 3V3
BME280 GND → STM32 GND
BME280 SCL → STM32 pin B6
BME280 SDA → STM32 pin B7
```

## Connection Details

### VIN
Powers the BME280 sensor using the STM32's 3.3 V supply.

### GND
Provides a common electrical reference between the STM32 and the BME280. Both devices must share the same ground to communicate correctly.

### SCL
The I²C clock line. The STM32 generates this clock to synchronize communication with the BME280.

### SDA
The I²C data line. Both the STM32 and the BME280 use this line to send and receive data.
